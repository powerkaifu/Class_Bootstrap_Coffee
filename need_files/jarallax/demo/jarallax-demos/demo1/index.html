Jarallax open source advanced javascript annimation library

What is Jarallax?

Jarallax is a open source javascript animation library. Jarallax makes it possible to easily manipulate css based on user input. For example with the scroll bar.

For who is it?

Jarallax is for web designers who want to give their creations an extra dimension. It is very easy with Jarallax to create parallax scrolling websites like ben the bodyguard and Nike Better World
